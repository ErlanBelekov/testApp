# testApp

<h1>Project setup</h1>

First, clone the project:
```git clone https://github.com/ErlanBelekov/testApp.git```

<h2>Now follow platform-specific instructions:</h2>

<h2>IOS</h2>

```
  cd testApp
  yarn
  cd ios
  pod install
  cd ..
  npx react-native run-ios
```

<h2>Android</h2>

```
  cd testApp
  yarn
  npx react-native run-android
```
